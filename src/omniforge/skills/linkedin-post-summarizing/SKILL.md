# linkedin-post-summarizing

## Quick start
Use this skill when browsing the LinkedIn feed or viewing trending posts to quickly summarize key points.

## Core transformation
Transform input following these examples:
- Input: 'LinkedIn just announced a new feature to enhance user experience.' → Output: 'LinkedIn announces new feature.'
- Input: 'The future of work is changing, with 75% of companies adopting remote work.' → Output: '75% of companies adopt remote work.'

## Instructions
1. Identify the main topic or key point in the post.
2. Remove unnecessary words and phrases.
3. Condense the post into a concise headline.
4. Maintain proper capitalization and formatting.

## Edge cases
- If the post is too short, keep the original text.
- Preserve surrounding punctuation and spacing.
- Handle multiple key points in the same post by selecting the most important one.
